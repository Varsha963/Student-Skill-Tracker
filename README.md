# Student Skill Tracker

## Project Description

Student Skill Tracker is a Core Java console-based application that helps students manage and track their technical skills, learning progress, and certifications.

## Features

- Add Skill
- View Skills
- Update Skill
- Delete Skill
- Search Skill
- View Completed Skills

## Technologies Used

- Core Java
- Object-Oriented Programming (OOP)
- ArrayList & Collection
- Exception Handling
- VS Code

## Project Structure

StudentSkillTracker/

├── src/
│   ├── Main.java
│   ├── Skill.java
│   └── SkillManager.java

## How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Compile the Java files:

javac *.java

4. Run the project:

java Main
